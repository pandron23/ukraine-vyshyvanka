# 🎨 Інструкція користувача: Інтерактивний додаток «Урок Вишиванки» 🇺🇦

Цей веб-додаток створений спеціально для того, щоб зробити ваш урок про українську вишиванку захопливим, сучасним та інтерактивним! 

Учні зможуть відчути себе справжніми майстрами та вишити свій перший оцифрований орнамент **хрестиком** на екрані смартфона, а ви зможете зібрати їхні роботи в єдину спільну галерею вишиванок на великому екрані!

---

## 📋 Що потрібно для запуску?

1. **Ноутбук вчителя** (на якому буде запущено програму).
2. **Смартфони учнів** (для малювання).
3. **Локальна мережа Wi-Fi**. 
   * *Дуже важливо:* Ноутбук вчителя та телефони учнів мають бути підключені до **однієї й тієї самої мережі**. 
   * **Порада:** Якщо у школі немає стабільного Wi-Fi, ви можете за 10 секунд увімкнути **"Точку доступу" (мобільний хотспот)** на своєму телефоні та підключити до неї свій ноутбук та телефони учнів. Інтернет-трафік при цьому практично не використовується!

---

## 🚀 Кроки для запуску програми

### Крок 1. Перевірка Node.js
Переконайтеся, що на вашому ноутбуці встановлено безкоштовну платформу **Node.js**:
* Якщо її немає, завантажте та встановіть її в один клік з офіційного сайту: [https://nodejs.org/](https://nodejs.org/) (виберіть версію **LTS**).

### Крок 2. Встановлення залежностей (робиться один раз)
1. Відкрийте папку з проєктом (`ukraine vsyvanka`).
2. Відкрийте консоль (Термінал / PowerShell / Командний рядок) у цій папці.
3. Виконайте команду для встановлення необхідних модулів:
   ```bash
   npm install
   ```

### Крок 3. Запуск програми
1. У консолі виконайте команду для запуску сервера:
   ```bash
   npm start
   ```
2. Ви побачите вітальне повідомлення в консолі:
   ```text
   ==================================================
   🇺🇦 СЕРВЕР «УРОК ВИШИВАНКИ» ЗАПУСКАЄТЬСЯ...
   🔗 Адреса для підключення учнів: http://192.168.1.15:3000
   👨‍🏫 Адреса для входу вчителя: http://192.168.1.15:3000/teacher.html
   ==================================================
   ```
3. Сервер автоматично визначить IP-адресу вашого ноутбука в мережі (наприклад, `192.168.1.15`).

## 👨‍🏫 Як проводити урок?

1. **Відкрийте Панель Вчителя:**
   * На своєму ноутбуці відкрийте браузер і перейдіть за адресою: `http://localhost:3000/teacher.html` (або за адресою, вказаною в консолі).
   * Виведіть це вікно браузера на великий екран / проектор.

2. **Підключення учнів:**
   * На екрані з'явиться великий красивий **QR-код** та посилання.
   * Попросіть учнів дістати смартфони, відкрити камеру та відсканувати цей QR-код.
   * На телефонах учнів відкриється сайт, де вони мають ввести своє ім'я (наприклад, *Данило* або *Марічка*) та натиснути кнопку **"Приєднатися до уроку"**.
   * Щойно учень підключиться, ви миттєво побачите його ім'я на великому екрані у вигляді красивої анімованої картки!

3. **Запуск малювання:**
   * На панелі вчителя (ліворуч) виберіть тривалість сесії (наприклад, 60 або 120 секунд) та натисніть кнопку **"🚀 Запустити вишивання"**.
   * У цей же момент на екранах всіх учнів відкриється зручний редактор вишивки хрестиком на тлі контуру традиційної сорочки.
   * Учні пальцями обирають кольори на палітрі (червоний, чорний, синій, жовтий, зелений) та «вишивають» орнамент.

4. **Завершення роботи:**
   * Коли час закінчиться (або якщо ви натиснете кнопку **"🛑 Завершити зараз"** на своєму ноутбуці), малювання автоматично зупиниться.
   * Всі малюнки учнів будуть миттєво зібрані сервером та виведені на великий екран у вигляді приголомшливої віртуальної **виставки вишиванок нашого класу**!

5. **Розгляд та збереження результатів:**
   * Ви можете клікати на будь-яку вишиванку, щоб відкрити її у великому масштабі.
   * Натиснувши кнопку **"Зберегти вишиванку (PNG)"**, ви можете завантажити малюнок учня собі на комп'ютер як повноцінне зображення! Це чудовий спосіб зберегти пам'ять про цей урок або роздрукувати дитячі роботи.

## 💡 Корисні поради для вчителя
* **Кольори:** Розкажіть учням про значення кольорів перед початком малювання (червоний — любов, чорний — земля та смуток, синій — небо та вода, жовтий — сонце та жито).
* **Симетрія:** Поясніть, що традиційні українські геометричні орнаменти часто мають дзеркальну симетрію — це допоможе дітям створити ще гарніші візерунки.
* **Творчість:** Не обмежуйте дітей! Нехай кожен створить свій унікальний малюнок, який виражає його настрій.
